# winter2021_recognition

## detect_protective_equipment

### Install
`python3` version: `3.10.0`  
`pip3` version: `21.2.4`

```python
pip3 install -r PPE_vig_winter/detect_protective_equipment_requirements.txt
```

### Usage
```python
mask_detector = MaskDetector(image_path, confidence)
persons = mask_detector.run()
```
